# Vinted_Discord_Bot
A discord bot who communicate with Vinted API

---
# Installation

```
npm install
```

> You need to create a `.env` file in the folder
>
> The `.env` file look like this
>
>>```
>>TOKEN=""
>>CLIENT_ID=""
>>SECRET_ID=""
>>SERVER_ID=""
>>```

---
## Deploy commands on server

```
npm run deploy
```

---
## Start the client

```
npm start
```
